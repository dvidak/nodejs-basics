API

USERS

GET

Fetching all users
api/users
Fetching user with a given id
api/users/:id
Fetching all books by user with a given id
api/users/:id/books

POST

Register new user
api/users
Login user
api/login

PUT

Update username of existing user with a given id
api/users/:id

DELETE

Delete user with a given username
api/users/:username
